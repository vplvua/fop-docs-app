# D-011 — Створення локального клієнта без moeosbb_user_id

**Дата:** 2026-05-16

**Рішення:** В адмінці можна створити клієнта без посилання на запис у "Моє ОСББ" (`moeosbb_user_id = NULL`). Такі клієнти є повноцінними локально, sync їх не торкається. Пізніше можна вручну вписати `moeosbb_user_id`, щоб увімкнути sync.

**Альтернативи:**

- Жорстко вимагати наявність клієнта в "Моє ОСББ" перед створенням локально.

**Обґрунтування:** Можливі краєві випадки — клієнт ще не заведений у "Моє ОСББ", або є контрагенти, які не є клієнтами SaaS, але платять (історичні борги тощо). Власник не хоче блокувати створення акту через відсутність запису в основній БД.

**Наслідки:**

- Поле `moeosbb_user_id` nullable.
- Sync-логіка ігнорує клієнтів з NULL.
- Можлива неузгодженість, якщо потім такий клієнт з'явиться в "Моє ОСББ" з тим самим ЄДРПОУ — вирішується ручною прив'язкою (через адмінку), без автомагії.
